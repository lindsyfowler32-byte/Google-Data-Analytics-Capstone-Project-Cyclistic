## Cyclistic Bike-Share Analysis in R

A data analytics capstone project using Divvy bike-share data (Q1 2019 & Q1 2020). This analysis demonstrates data preparation, cleaning, aggregation, and visualization in R to compare behavioral differences between casual riders and annual members.

**Skills:** R · Data Cleaning · Data Analysis · Data Visualization · Tidyverse · Business Insights
